## Props

| 名称    |      类型      | 必填 | 可选值 | 默认值                  | 描述                       |
| :------ | :------------: | :--: | :----- | :---------------------- | :------------------------- |
| id      |     String     |      |        |                         | 组件id，若不设置会自动生成 |
| display |     Boolean    |      |        | true                    | 显示隐藏                   |
| value   | [String, Date] |      |        | <pre>"new Date()"</pre> | 初始化日期                 |
